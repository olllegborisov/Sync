## Помнить во время верстки

-  Aria-* атрибуты 
	  aria-label="Основная навигация" 
	  aria-label="Меню"
 - Для всех внешних ссылок ведущих на другие сайты rel="nofollow"
 - Для видео preload="none"
- Загрузка изображение loading="lazy", не использовать на первых секциях страниц
- В карточке товара, если есть выбор цвета, спросить нужно ли делать  цвета через inline-stlye
- Для контентной части делать внешний тэг с классом. И с помощью внешнего класса ссылаться на внутренние тэги.
- Во время коммита запускаются [[Линтеры |линтеры]]. Внутри ссылки указаны проверки которые задействует EsLInt
- Форме классы не задавать. При необходимости сделать обертку инпутам.
- Фильтры делать через форму [пример](https://bitrix.dart-s.ru/catalog/shoes/](https://bitrix.dart-s.ru/catalog/shoes/ "https://bitrix.dart-s.ru/catalog/shoes/")).
- Задавать отступы с помощью тильды ~ , что бы не использовать last-child. Так же использовать тильду для создания элементам всем кроме первых.
- Для повторяющихся элементов через раз использовать nth-child(even), nth-child(2n + 1) 
- Использовать псевдоэлементы с svg, когда требуется масштабируемость верстки. Как при использовании ~ или nth-child(even), nth-child(2n + 1) 
- Чем меньше самих классов. И чем они короче сами по себе тем лучше. Для бэка
