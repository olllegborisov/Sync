# Javascript файлы

## module

Универсальные блоки, не завязанные на конкретном DOM
###### Пример:
accordion, tabs, dropdown, fancybox

## components

Компоненты конкретны: они связывают модуль с определённой разметкой на странице. Или просто вызывают функцию модуля. Связаны с DOM деревом
###### Пример
header, slider, checkout-form, cart

## utils 
Эти утилиты вспомогательные функции. Не зависят от DOM. Не инициализируются сами по себе. Часто переиспользуются в разных модулях обычно работают с данными, строками, числами, массивами и т.д.
Если функция ничего не знает про страницу, не лезет в DOM, и делает какую-то общую логику - это utils
###### Пример
debounce, formatPrice, FormDate, isMobile, checkbrowser

# CSS файлы

base
	шрифты, переменные, миксины, анимации, reset, normalize
component
	Стили для компонентов (кнопки, попапы, табы и т.д)
includes
	Header и Footer
vendor
	Кастомные стили библиотек, если нужно (swiper, fancybox, panzoom и тд)
