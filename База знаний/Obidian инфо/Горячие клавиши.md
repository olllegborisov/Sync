
# Стандартные

- Создавать заметки лучше через ctrl + o 
- Ctrl + B — **Жирный шрифт** 
- Ctrl + I — *Курсив* 
- Ctrl + K — вставить ссылку


# Плагины
### Advanced tables

- ctrl + shift + D — открыть редактор таблиц


# Настроил сам 

- Ctrl + Alt + g — панель тэгов.
- Ctrl + F1 — выбор заголовка
- Ctrl + Shift + K — вставить внутреннюю ссылку





> [!info] Увидеть все горячие клавиши можно нажав на шестеренку, и на 'сочетания клавиш'
