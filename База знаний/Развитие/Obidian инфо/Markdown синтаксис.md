\ — обратный слэш экранирует действие кода внутри страницы. То есть работать код не будет, зато его можно будет прочитать, как ниже. И Символы будет видно. По умолчанию они скрываются.
Пример:
- [ ]  работающий checkbox
\- \[ \] — экранированный checkbox. Нажми на строку

## Ссылки

\[\[ ]] — внутренние ссылки
\[]() — внешние ссылки
\[\[Markdown|пример]] — якорная ссылка на [[Markdown]]. Позволяет видеть слово "пример", а при клике переходить на страницу [[Markdown]].


## Другое

\#expample — теги 
\#example/subExample — под теги
\# Example — заголовки
\*\*text** — жирный шрифт
\- \[ \] — checkbox
