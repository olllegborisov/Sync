
## Установка на пк

1.  Создать приватный репозиторий.
2. [Создать токен]([https://github.com/settings/tokens/new](https://github.com/settings/tokens/new)) ,поставить галку на repo (и вложенности) если нет. (нужен при первом репозитории, заработал пуш и без него во втором)
3. Скопировать токен
4. Установить плагин git на ПК
